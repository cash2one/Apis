核心操作说明：

	此数据结构是采用无限分类的方式
	level字段值解释：
		1：全部的省数据
		2:全部的市数据
		3:全部的县数据
		4:全部的镇数据
	pid字段解释：
		无限分类结构必须的字段，不懂的请自行搜索查询

正常使用方式如下：

	先获取一级数据，两种方法
		select * from 你的表名 where level=1；//获取省数据
		select * from 你的表名 where pid=0；//获取省数据

	然后再获取当前省下面的市时，
		select * from 你的表名 where pid=省id

	获取县镇等操作，以此类推
		select * from 你的表名 where pid=市id或县id
